# Mock Interview Coach

## Overview
An intelligent Python mock interview coach that generates interview questions, evaluates answers, checks for role-specific keywords, and gives feedback.

## Features
- Career field selection
- Random interview questions
- Answer scoring
- Keyword matching
- Personalized feedback
- Final interview performance report

## Career Categories
- Data Analyst
- Healthcare
- Banking

## Skills Demonstrated
- Python
- Functions
- Dictionaries
- Lists
- Random Module
- Loops
- User Input
- Text Analysis
- Conditional Logic
